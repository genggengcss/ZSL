== Installation ==
Download "minFunc" package (https://www.cs.ubc.ca/~schmidtm/Software/minFunc.html), put it under the folder /SynC, and compile.
Check /SynC/codes/set_path.m, which is called in /SynC/codes/SynC.m, to make sure the path is correct.
